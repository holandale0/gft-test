-------------------------------------------------------------------------------------------------------
INFORMAÇÕES TÉCNICAS
-------------------------------------------------------------------------------------------------------

 - Versão do Java: 8
 - Package principal : main
 - Package de testes : test
 - Classe Principal : GftDeveloperPracticum.java
 - Classe de Testes : GftDeveloperPracticumTest.java

-------------------------------------------------------------------------------------------------------
INSTRUÇÕES
-------------------------------------------------------------------------------------------------------

1 - Execução da aplicação

	1.1 - Abir o terminal(Linux) ou cmd(Windows)
	1.2 - Ir até o diretório teste-gft\src\main
	1.3 - Gerar o .class através do comando: javac GftDeveloperPracticum.java
	1.4 - Retornar um diretório (teste-gft\src)
	1.5 - Executar a classe GftDeveloperPracticum através do comando: java main.GftDeveloperPracticum
	1.6 - Informar os parâmetros. Ex: night,1,3,2,1
	1.7 - Teclar Enter


2 - Execução dos testes

	2.1 - Abir o terminal(Linux) ou cmd(Windows)
	2.2 - Ir até o diretório teste-gft\src\test
	2.3 - Gerar o .class através do comando: javac GftDeveloperPracticumTest.java
	2.4 - Retornar um diretório (teste-gft\src)
	2.5 - Executar a classe GftDeveloperPracticum através do comando: java test.GftDeveloperPracticumTest
